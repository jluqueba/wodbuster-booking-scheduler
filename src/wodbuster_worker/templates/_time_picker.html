{# 24-hour time picker macro (Flatpickr). Renders a single text that Flatpickr enhances into a compact hour/minute popover. Configuration: - ``time_24hr: true`` forces 24h regardless of browser/OS locale (this is the whole reason we swapped the native widget out). - ``noCalendar: true`` + ``enableTime: true`` yields the time-only floating panel. - ``dateFormat: 'H:i'`` pins the wire value to ``HH:MM``, which is exactly what the form parser expects — zero server-side normalisation. - Clock icon is a background SVG on the input itself so it moves with focus/hover states. The macro emits both the visible input and a hidden fallback input holding the same value. If Flatpickr fails to load (CDN outage, blocked script), the plain input still works: the operator can type HH:MM by hand and the server accepts it. Usage: {% import "_time_picker.html" as widgets %} {{ widgets.time_picker("class_time", form_values.get("class_time"), required=true) }} #} {% macro time_picker(name, value, required=false) -%} {% set safe_id = name | replace('_', '-') %}
{%- endmacro %}
