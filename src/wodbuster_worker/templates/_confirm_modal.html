{# Reusable custom confirm modal (wbConfirm). Renders a hidden dialog + inline script. Any form that wants to ask for confirmation before submitting swaps its ``onsubmit="return confirm('...')"`` handler for ``onsubmit="return wbConfirm(this, event, '...')"``. The function returns ``false`` on first invocation, opens the modal, and re-submits the form when the operator taps Confirm. Uses the native element (baseline available in every modern browser as of 2023+); no external JS dependency. Falls back to the browser confirm() when is missing. #}
